## Decompiler

From [this issue](https://github.com/NationalSecurityAgency/ghidra/issues/720) some indication on how to debug the decompiler

```
$ cd Ghidra/Feature/Decompiler/src/decompile/cpp
$ make decomp_dbg
$ export SLEIGHHOME=~/git/ghidra/
$ ./decomp_dbg 
[decomp]>
```
